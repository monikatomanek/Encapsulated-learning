# OOP Python Guide

This folder is part of the `Encapsulated-learning` repository by [Monika Tomanek](https://github.com/monikatomanek) and contains a comprehensive, beginner-friendly learning guide to Object-Oriented Programming (OOP) in Python.

## Contents

* `oop_guide.md` - A detailed, descriptive guide that walks through each major OOP principle in Python, including:

  * What is OOP?
  * Classes and Objects
  * Encapsulation
  * Inheritance
  * Polymorphism
  * Abstraction
  * Class and Static Methods

* `sample_code/` *(coming soon)* - Real-life example projects and Python scripts demonstrating each OOP concept in action.
