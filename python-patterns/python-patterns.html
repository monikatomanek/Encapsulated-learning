Core Patterns

1. Load a CSV

df = pd.read_csv('filename.csv')
'Bring my spreadsheet into Python and call it df'

2. Count values in a column

df['ColumnName'].value_counts()
'How often does each unique value appear in this column?'

3. Count values as percentages

df['ColumnName'].value_counts(normalize=True) * 100
'What percent of the total is each value in this column?'

4. Filter rows by condition

filtered_df = df[df['ColumnName'] == 'SomeValue']
'Give me only the rows where this column has this value'

5. Find the most common value


top_result = df['ColumnName'].value_counts().head(1)
top_name = top_result.index[0]
top_value = top_result.values[0]
'Whatâ€™s the most common category in this column?'

6. Format output nicely

print(f"\nPercentage: {value:.2f}%")
'Say this sentence and plug in my number using 2 decimal places'

7. Boolean filter (True/False check)

df[df['BooleanColumn'] == True]
'Give me only the rows where this is True'

8. Conditional logic (if / elif / else)

if score >= 90:
    print("Excellent")
elif score >= 75:
    print("Good")
else:
    print("Needs improvement")

9. Loop through values

for item in df['ColumnName'].unique():
    print(item)

10. Create a new column from logic

def custom_logic(row):
    if row['Column'] == 'something':
        return 'A'
    return 'B'

df['NewCol'] = df.apply(custom_logic, axis=1)

11. Group stats (mean, sum, etc.)

df.groupby('GroupCol')['TargetCol'].mean()
'Group by this category, then average or sum something'
